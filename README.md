# 網路測試工具 🌐

一個功能完整的網路連線品質檢測工具，提供繁體中文介面，支援多種網路測試功能。

## ✨ 功能特色

### 🚀 核心測試功能
- **網路速度測試** - 檢測下載/上傳速度和延遲
- **抖動測試** - 分析網路連線穩定性
- **封包遺失測試** - 檢測網路封包遺失率
- **伺服器連線測試** - 測試多個地區伺服器連線狀況
- **DNS 解析測試** - 檢測域名解析效能
- **完整測試報告** - 一鍵執行所有測試並生成詳細報告

### 🔍 用戶資訊檢測
- 自動檢測用戶 IP 位址
- 顯示網路供應商 (ISP) 資訊
- 地理位置資訊
- 連線類型識別 (2G/3G/4G/5G)

### 📊 測試結果
- 即時顯示測試進度
- 彩色狀態指示器 (良好/警告/差劣)
- 詳細的測試數據分析
- 整體網路品質評分 (0-100)

### 💾 報告功能
- 匯出 JSON 格式測試報告
- 一鍵分享測試結果
- 支援系統原生分享 API
- 自動複製到剪貼簿

## 🚀 快速開始

### 線上使用
直接訪問: [https://onx999.github.io/network-test-tool](https://onx999.github.io/network-test-tool)

### 本地運行
```bash
git clone https://github.com/ONX999/network-test-tool.git
cd network-test-tool
python -m http.server 8000
# 開啟瀏覽器訪問 http://localhost:8000
```

## 🎯 測試項目

- **速度測試**: 下載/上傳速度和延遲測試
- **抖動測試**: 20次連續測試分析穩定性
- **封包遺失**: 100個封包的遺失率檢測
- **伺服器測試**: 多地區伺服器連線測試
- **DNS解析**: 域名解析效能測試

## 🛠️ 技術特色

- 純前端實現，無需後端伺服器
- 響應式設計，支援各種裝置
- 繁體中文介面
- 現代化 UI 設計
- 自動 GitHub Pages 部署

## 📱 使用方式

1. 開啟網頁應用程式
2. 等待自動檢測用戶資訊
3. 選擇測試項目或執行完整測試
4. 查看詳細結果和評分

### 快捷鍵
- `Ctrl + 1-5`: 個別測試
- `Ctrl + Enter`: 完整測試
